# Model Advisor

ModelAdvisor is deterministic and advisory. It classifies the user prompt into categories such as quick question, planning, testing, security review, web research, and full build cycle.

It recommends:
- Workflow
- Agents and roles
- Model tier
- Context level
- Safety mode
- Scout usage when current information is requested

Deep reasoning recommendations require confirmation when configured. Empty concrete model mappings fall back to provider defaults.
