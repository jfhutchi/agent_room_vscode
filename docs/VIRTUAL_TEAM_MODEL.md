# Virtual Team Model

Virtual agents are named participants. They are not the same as providers.

Default agents:
- User uses `human`.
- Atlas and Forge use `claudeCodeCli`.
- Sentinel and Gauge use `codexCli`.
- Scout uses `openAiWebSearch` and is disabled by default.
- Conductor uses `internalConductor`.

The UI shows shared provider usage pools so users understand that multiple agents can consume the same local CLI account and quota.
