# 002 Build Professional Webview UI

Build the Agent Room webview with plain HTML, CSS, and JavaScript. Use VS Code CSS variables, strict CSP, nonced scripts, no remote assets, no external fonts, and no unsafe HTML insertion for profile or transcript data.

Required UI:
- Header, provider health strip, workflow/safety controls.
- Collapsible room setup with team cards and role matrix.
- Transcript with compact messages and diagnostics expanders.
- Composer with context chips and send/workflow/export/stop actions.
